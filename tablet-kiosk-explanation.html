01. ISSUE現場の課題
CURRENT ISSUE

現場のタブレット運用、こんな課題はありませんか

介護現場ではIT機器の標準化が難しく、「使える人」と「使えない人」の差が業務品質に直結します。汎用タブレットの導入だけでは、運用負荷がむしろ増えるケースも少なくありません。

CHALLENGE 01

ITリテラシーのばらつき

スタッフの年齢層が幅広く、ホーム画面・アプリ切替・通知の概念がわからない方も多い。新人教育に毎回時間を要する。

CHALLENGE 02

業務外アプリの起動・誤操作

SNS・カメラ・ブラウザなどを開いてしまう、設定を変更してしまう。元に戻す対応で本部・施設長の工数が消費される。

CHALLENGE 03

紛失・私物化・情報漏えい

端末の所在管理が曖昧で、紛失時の入居者情報漏えいリスクが残る。アップデートやアプリ配信も属人化しがち。

04. ARCHITECTURE技術構成
TECHNOLOGY

4つの技術で「使い手を選ばない端末」を実現

標準化された端末管理技術を組み合わせ、業務に必要な機能だけを安全に提供します。

MDM管理

本部から全端末を一括管理。アプリ配信・OS更新・ポリシー変更を遠隔で実施。

ガイドアクセス / Kiosk Lock

iOSガイドアクセスまたはAndroid Enterpriseで他アプリへの離脱を物理的に遮断。

PWA配信

TELLURはPWA（Webアプリ）。バージョン更新は配信のみで完結し、再配布不要。

遠隔ロック・消去

紛失時は本部から即時ロック・データ初期化。位置情報の追跡も可能。

通信構成: 専用タブレット ⟷ MDMサーバー（ポリシー・遠隔操作） ⟷ TELLURクラウド（業務データ・AI推論） ／ HTTPS / TLS1.3 / E2E暗号化

08. SECURITYセキュリティと運用
SECURITY & OPERATIONS

本部からの一元管理で、運用負荷を最小化

情報漏えい・端末紛失・アップデート対応といった「IT管理者がいない現場」の不安を、すべて本部で巻き取る設計です。

SECURITY 01

紛失時のリモートロック・初期化

端末を紛失した場合、本部ダッシュボードから即座にロック・データ消去・位置追跡が可能。入居者情報の漏えいを防止します。

SECURITY 02

アップデートの一元配信

OS・TELLURアプリの更新は本部が一括管理。施設側で「更新作業を促す」必要は一切ありません。バージョン揃えも自動。

SECURITY 03

監査ログ・アクセス記録

誰がいつどの端末でどの記録を入力・閲覧したかを自動記録。事故・トラブル時の事実確認が可能。介護記録の信頼性を担保。

SECURITY 04

通信の暗号化・閉域接続オプション

すべての通信はTLS1.3で暗号化。エンタープライズプランでは閉域SIM・VPN接続による完全閉域運用にも対応します。
